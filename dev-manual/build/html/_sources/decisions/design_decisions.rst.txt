Design Decisions
================

Introduction
------------

All key architecture and design decisions taken during the course of the project will be documented here.

By documenting decisions in this way, they can easily be communicated to a wide audience and the context for a decision remains intact for the duration of the project (and beyond)

Each decisions should be described using:
 * A short name
 * A description of the decision
 * An explanation of the rationale for the decision
 * A description of the impact of the decision
